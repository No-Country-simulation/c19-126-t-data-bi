- Librer칤as Python: Text Blob

La polaridad determina el sentimiento del texto. Sus valores se encuentran en [-1,1] donde -1 denota un sentimiento muy negativo y 1 denota un sentimiento muy positivo.
La subjetividad determina si un ingreso de texto es informaci칩n objetiva o una opini칩n personal. Su valor se encuentra entre [0,1] donde un valor m치s cercano a 0 denota una informaci칩n f치ctica y un valor m치s cercano a 1 denota una opini칩n personal.[Referencia](https://www.analyticsvidhya.com/blog/2022/07/sentiment-analysis-using-python/)
